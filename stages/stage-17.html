🟢 ۱ تا ۷: ساده (گرم کردن)

🟢 تمرینات سطح ساده (۱ تا ۷)

۱
بررسی بخش‌پذیری بر ۵
👆 کلیک کن
✅ پاسخ:
n = int(input("عدد: ")) if n % 5 == 0: print("بخش‌پذیر بر ۵") else: print("بخش‌پذیر نیست")
۲
بزرگترین عدد بین سه عدد
👆 کلیک کن
✅ پاسخ:
a=int(input()); b=int(input()); c=int(input()) m = a if b > m: m = b if c > m: m = c print(m)
۳
چاپ اعداد فرد ۱ تا ۲۰
👆 کلیک کن
✅ پاسخ:
for i in range(1, 21, 2): print(i)
۴
جدول ضرب عدد دلخواه
👆 کلیک کن
✅ پاسخ:
n = int(input("عدد: ")) for i in range(1, 11): print(f"{n}×{i}={n*i}")
۵
میانگین ۵ عدد
👆 کلیک کن
✅ پاسخ:
s = 0 for i in range(5): s += float(input()) print(s/5)
۶
بررسی سال کبیسه
👆 کلیک کن
✅ پاسخ:
y = int(input()) if y%400==0 or (y%4==0 and y%100!=0): print("کبیسه") else: print("عادی")
۷
شمارش معکوس با while
👆 کلیک کن
✅ پاسخ:
n = 10 while n > 0: print(n) n -= 1 print("GO!")

🟡 تمرینات سطح متوسط (۸ تا ۱۴)

۸
محاسبه توان بدون **
👆 کلیک کن
✅ پاسخ:
b=int(input("پایه: ")) e=int(input("توان: ")) r=1 for i in range(e)
✅ پاسخ:
b=int(input("پایه: ")) e=int(input("توان: ")) r=1 for i in range(e): r*=b print(r)
۹
عدد اول بودن
👆 کلیک کن
✅ پاسخ:
n=int(input()); p=True if n<2: p=False for i in range(2,int(n**0.5)+1): if n%i==0: p=False;break print("اول" if p else "اول نیست")
۱۰
مجموع ارقام یک عدد
👆 کلیک کن
✅ پاسخ:
n=int(input()); s=0 while n>0: s+=n%10 n//=10 print(s)
۱۱
معکوس یک عدد
👆 کلیک کن
✅ پاسخ:
n=int(input()); r=0 while n>0: r=r*10+n%10 n//=10 print(r)
۱۲
ب.م.م دو عدد
👆 کلیک کن
✅ پاسخ:
a=int(input()); b=int(input()) while b!=0: a,b=b,a%b print(a)
۱۳
الگوی مثلثی با ستاره
👆 کلیک کن
✅ پاسخ:
n=int(input()) for i in range(1,n+1): print("*"*i)
۱۴
تشخیص عدد کامل
👆 کلیک کن
✅ پاسخ:
n=int(input()); s=0 for i in range(1,n): if n%i==0: s+=i print(s==n)

🔴 تمرینات سطح چالشی (۱۵ تا ۲۰)

۱۵
سری فیبوناچی
👆 کلیک کن
✅ پاسخ:
n=int(input()); a,b=0,1 for i in range(n): print(a,end=" ") a,b=b,a+b
۱۶
لوزی توخالی
👆 کلیک کن
✅ پاسخ:
n=int(input()) for i in range(1,n+1): s=" "*(n-i)+"*" if i>1: s+=" "*(2*i-3)+"*" print(s) for i in range(n-1,0,-1): s=" "*(n-i)+"*" if i>1: s+=" "*(2*i-3)+"*" print(s)
۱۷
مثلث خیام-پاسکال
👆 کلیک کن
✅ پاسخ:
n=int(input()) for i in range(n): c=1 print(" "*(n-i),end="") for j in range(i+1): print(c,end=" ") c=c*(i-j)//(j+1) print()
۱۸
تخمین عدد پی
👆 کلیک کن
✅ پاسخ:
n=int(input()); p=0 for i in range(1,n+1): p+=1/(i**2) print((p*6)**0.5)
۱۹
عدد آرمسترانگ
👆 کلیک کن
✅ پاسخ:
n=int(input()); t=n; s=0; d=len(str(n)) while t>0: s+=(t%10)**d; t//=10 print(s==n)
۲۰
رمزنگاری سزار
👆 کلیک کن
✅ پاسخ:
t=input("متن: "); s=int(input("شیفت: ")); r="" for ch in t: if ch.isalpha(): b=65 if ch.isupper() else 97 r+=chr((ord(ch)-b+s)%26+b) else: r+=ch print(r)

۵ سوال برای سنجش نهایی (اختیاری)

۲. کدام حلقه برای «تا وقتی کاربر نگفته quit ادامه بده» مناسب است؟
۳. break در حلقه while چه می‌کند؟
۴. برای بررسی زوج بودن از کدام عملگر استفاده می‌شود؟
