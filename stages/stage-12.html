🔄 تبدیل انواع داده (Type Casting)

یاد می‌گیری چطور نوع داده‌ها را به هم تبدیل کنی

👋 سلام، دانشجو جان! امروز تبدیل داده‌ها رو یاد می‌گیریم 🔄

🔄 تبدیل نوع داده یعنی چه؟

تا حالا فهمیدی که "123" (رشته) با 123 (عدد) فرق داره. اما گاهی مجبوری یکی را به دیگری تبدیل کنی! مثلاً وقتی با input() از کاربر عدد می‌گیری، ورودی همیشه رشته است و برای انجام محاسبات باید آن را به عدد تبدیل کنی.

به این کار می‌گویند Type Casting یا تبدیل نوع داده. یعنی تغییر نوع یک داده از یک فرم به فرم دیگر.

🎯 تعریف ساده: Type Casting یعنی «جنسیت» یک داده را عوض کنی! مثل اینکه یک عدد را به متن تبدیل کنی، یا یک متن را به عدد. پایتون توابع آماده‌ای برای این کار دارد.

📋 توابع اصلی تبدیل نوع داده

پایتون چهار تابع اصلی برای تبدیل انواع داده دارد:

تابعکاربردورودیخروجیمثال
int()تبدیل به عدد صحیحرشته عددی یا اعشاریعدد صحیحint("42") → 42
float()تبدیل به عدد اعشاریرشته عددی یا صحیحعدد اعشاریfloat("3.14") → 3.14
str()تبدیل به رشتههر نوع داده‌ایرشتهstr(42) → "42"
bool()تبدیل به بولینهر نوع داده‌ایTrue/Falsebool(1) → True

🔢 تبدیل به عدد صحیح با int()

تابع int() هر چیزی را که بتواند به عدد صحیح تبدیل کند، تبدیل می‌کند. رشته‌های عددی و اعداد اعشاری را می‌توانی به int تبدیل کنی.

print(int("42") + 8) print(int(3.99)) age = int(input("سن تو: ")) print(f"سال دیگه {age + 1} ساله می‌شی!")
50 3 سال دیگه 21 ساله می‌شی!
⚠️ هشدار: int("Hello") خطای ValueError می‌دهد! فقط رشته‌هایی را می‌توانی به int تبدیل کنی که واقعاً عدد باشند.

📏 تبدیل به عدد اعشاری با float()

تابع float() داده را به عدد اعشاری تبدیل می‌کند. هم رشته‌های عددی و هم اعداد صحیح را می‌توانی به float تبدیل کنی.

pi = float("3.14159") print(pi * 2) print(float(10)) price = float(input("قیمت: ")) discount = price * 0.9 print(f"قیمت با تخفیف: {discount}")
6.28318 10.0 قیمت با تخفیف: 90000.0

📝 تبدیل به رشته با str()

تابع str() هر نوع داده‌ای را به رشته تبدیل می‌کند. این خیلی کاربردی است وقتی می‌خواهی یک عدد را در کنار متن قرار دهی!

num = 42 text = str(num) print("عدد " + text) status = str(True) print("وضعیت: " + status)
عدد 42 وضعیت: True

✅ تبدیل به بولین با bool()

تابع bool() هر مقداری را به True یا False تبدیل می‌کند. قانون کلی: مقادیر خالی یا صفر False می‌شوند، بقیه True.

print(bool(1)) # True print(bool(0)) # False print(bool("Hi")) # True print(bool("")) # False
True False True False

⚠️ خطاهای رایج در تبدیل نوع داده

خطاعلتمثالراه‌حل
ValueErrorتبدیل رشته غیرعددی به عددint("Hello")مطمئن شو رشته واقعاً عدد است
TypeErrorجمع رشته و عدد بدون تبدیل"Age: " + 25از str() استفاده کن
ValueErrorتبدیل رشته با کاراکتر اضافهint("42abc")رشته باید فقط عدد باشد

🧪 تمرین‌های عملی

روی هر کارت کلیک کن تا پاسخ را ببینی! اول خودت حل کن، بعد جواب را چک کن!

۱
رشته "100" را به عدد تبدیل کن و با 50 جمع بزن
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
num = int("100") result = num + 50 print(result) # 150
۲
عدد 25 را به رشته تبدیل کن و با متن ترکیب کن
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
age = 25 text = "سن: " + str(age) print(text) # سن: 25
۳
از کاربر دو عدد بگیر و حاصل جمع را چاپ کن
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
a = int(input("اول: ")) b = int(input("دوم: ")) print("جمع:", a + b)
۴
اعشاری 7.89 را به int تبدیل کن
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
x = int(7.89) print(x) # 7 (حذف اعشار)
۵
رشته خالی و عدد 0 را به bool تبدیل کن
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
print(bool("")) # False print(bool(0)) # False print(bool("Hi")) # True
۶
از کاربر وزن و قد بگیر و BMI محاسبه کن
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
w = float(input("وزن: ")) h = float(input("قد: ")) bmi = w / (h ** 2) print(f"BMI: {bmi:.2f}")
