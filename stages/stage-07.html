📦 متغیرها (Variables) در پایتون

یاد می‌گیری چطور اطلاعات رو توی برنامه ذخیره کنی و بعداً استفاده کنی!

👋 سلام، دانشجو جان! امروز با جعبه‌های جادویی پایتون آشنا میشیم 📦

📦 متغیر یعنی چی؟ بیا با یه مثال ساده بفهمیم!

تصور کن یه کمد لباس توی خونه داری. این کمد چندتا کشو داره. روی هر کشو یه برچسب زدی: «جوراب»، «تی‌شرت»، «شلوار». حالا هر وقت جوراب خواستی، لازم نیست کل خونه رو بگردی! فقط کافیه بری سراغ کشویی که روش نوشته «جوراب».

🗄️

متغیر دقیقاً مثل کشوی برچسب‌داره!

اسم متغیر = همون برچسب روی کشو
مقدار متغیر = چیزایی که داخل کشو می‌ذاری
هر وقت خواستی ازش استفاده کنی، فقط اسمش رو صدا می‌زنی!

متغیر (Variable) یه جعبه یا ظرف توی حافظه کامپیوتره که می‌تونی یه مقدار توش بذاری، بعداً تغییرش بدی، و هر وقت خواستی ازش استفاده کنی. اسمش رو هم خودت انتخاب می‌کنی!

🎯 تعریف ساده: Variable = یک اسم برای یک مقدار. مثل اینکه به عدد 25 بگی age (سن). از این به بعد هرجا بنویسی age، پایتون می‌فهمه منظورت 25 هست!

✍️ چطور یه متغیر بسازیم؟

ساختن متغیر در پایتون خیلی آسونه! فقط کافیه:

  1. یه اسم انتخاب کنی
  2. علامت مساوی (=) بذاری
  3. مقدار مورد نظر رو بنویسی
# ساخت یک متغیر به اسم name و ریختن مقدار "Ali" توش name = "Ali" # ساخت یک متغیر به اسم age و ریختن عدد 20 توش age = 20 # ساخت یک متغیر به اسم height و ریختن عدد اعشاری 1.75 توش height = 1.75 # حالا می‌تونیم این متغیرها رو چاپ کنیم! print(name) print(age) print(height)

خروجی:

Ali 20 1.75
🧠 نکته مهم: علامت = در پایتون به معنی «مساوی بودن» نیست! به معنی «برو مقدار سمت راست رو بریز توی متغیر سمت چپ» هست. همیشه اول سمت راست محاسبه میشه، بعد نتیجه می‌ره توی متغیر سمت چپ.

📝 قوانین نام‌گذاری متغیرها

برای اسم متغیرها باید حتماً این قوانین رو رعایت کنی، وگرنه پایتون خطا میده:

✅ مجاز❌ غیرمجازعلت
name1nameاسم متغیر با عدد شروع نمی‌شود
my_namemy-nameفقط از _ (آندرلاین) استفاده کن، خط تیره ممنوع!
age2age 2اسم متغیر نمی‌تواند فاصله داشته باشد
userNameclassاز کلمات کلیدی پایتون استفاده نکن
_private@emailفقط حروف، اعداد و _ مجاز است

خلاصه قوانین:

  • ✅ می‌تونه شامل حروف انگلیسی (a-z, A-Z)، اعداد (0-9) و آندرلاین (_) باشه
  • ❌ نمی‌تونه با عدد شروع بشه
  • ❌ نمی‌تونه فاصله یا کاراکتر خاص داشته باشه
  • ❌ نمی‌تونه از کلمات کلیدی پایتون باشه (مثل print, if, for, class)
  • ⚠️ پایتون به بزرگ و کوچک بودن حروف حساسه! (name و Name و NAME سه متغیر متفاوت هستن!)

🔄 تغییر مقدار متغیر

همونطور که می‌تونی محتویات کشو رو عوض کنی، می‌تونی مقدار متغیر رو هم تغییر بدی. مقدار جدید جایگزین مقدار قبلی میشه. مقدار قدیمی برای همیشه از دست می‌ره!

# اول یک متغیر می‌سازیم score = 10 print(score) # خروجی: 10 # حالا مقدارش رو تغییر می‌دیم score = 25 print(score) # خروجی: 25 (مقدار 10 دیگه وجود نداره!) # می‌تونیم از خود متغیر برای تغییر خودش استفاده کنیم! score = score + 5 print(score) # خروجی: 30 (25 + 5)
💡 نکته جالب: پایتون اول سمت راست = رو کامل محاسبه می‌کنه، بعد نتیجه رو می‌ریزه توی متغیر سمت چپ. برای همین می‌تونی بنویسی score = score + 5! پایتون میگه: خب ببینم score الان چنده؟ 25. با 5 جمع کن میشه 30. حالا 30 رو بریز توی score.

🧪 انواع داده‌هایی که می‌شه توی متغیر ریخت

توی پایتون، متغیرها می‌تونن هر نوع داده‌ای رو توی خودشون نگه دارن:

# 1. رشته (String) - متن message = "Hello Python!" country = 'Iran' # 2. عدد صحیح (Integer) - اعداد بدون اعشار students = 30 temperature = -5 # 3. عدد اعشاری (Float) - اعداد با اعشار pi = 3.14159 price = 99.99 # 4. بولین (Boolean) - فقط True یا False is_student = True is_raining = False

🖨️ استفاده از متغیرها با Print

حالا که متغیر داری، می‌تونی باهاشون توی print کار کنی:

# تعریف متغیرها name = "Sara" age = 22 city = "Tehran" # چاپ مستقیم متغیر print(name) # خروجی: Sara # ترکیب متن و متغیر با کاما print("Name:", name, "Age:", age) # خروجی: Name: Sara Age: 22 # ترکیب با f-string (بهترین روش!) print(f"My name is {name} and I am {age} years old") # خروجی: My name is Sara and I am 22 years old # محاسبه با متغیرها a = 15 b = 25 print(a + b) # خروجی: 40

🎯 چرا متغیرها اینقدر مهم هستن؟

بدون متغیر، برنامه‌نویسی تقریباً غیرممکنه! ببین چقدر کاربرد دارن:

  • 📊 ذخیره اطلاعات کاربر: اسم، سن، ایمیل، رمز عبور
  • 🧮 انجام محاسبات: جمع، تفریق، میانگین، فرمول‌های ریاضی
  • 🎮 بازی‌ها: امتیاز (score)، تعداد جان‌ها (lives)، سطح (level)
  • 🛒 فروشگاه: قیمت، تعداد، مجموع خرید، تخفیف
  • 📱 اپلیکیشن‌ها: تنظیمات کاربر، پیام‌ها، نوتیفیکیشن‌ها

⚠️ خطاهای رایج در کار با متغیرها

نام خطاعلتراه‌حل
NameError استفاده از متغیر قبل از تعریف اول متغیر رو با = تعریف کن، بعد استفاده کن
NameError غلط املایی در اسم متغیر اسم متغیر رو چک کن. مثلاً name رو اشتباهی nmae نوشتی!
NameError بزرگ و کوچک بودن حروف Name با name فرق داره! حواست به حروف بزرگ و کوچک باشه
SyntaxError شروع اسم متغیر با عدد مثلاً 1name غلطه، بنویس name1 یا first_name
SyntaxError استفاده از فاصله یا کاراکتر خاص بجای my name بنویس my_name، بجای user@name بنویس username
⚠️ اشتباه خیلی رایج: فراموش کردن کوتیشن برای رشته‌ها!
name = Ali ❌ غلط! (پایتون فکر می‌کنه Ali یه متغیره)
name = "Ali" ✅ درست! (پایتون می‌فهمه Ali یه رشته است)

✏️ نکات تکمیلی و ترفندهای متغیرها

🔹 تعریف چند متغیر در یک خط

# تعریف سه متغیر در یک خط x, y, z = 10, 20, 30 print(x, y, z) # خروجی: 10 20 30 # تعریف یک مقدار به چند متغیر a = b = c = 0 print(a, b, c) # خروجی: 0 0 0

🔹 جا به جا کردن مقادیر دو متغیر

first = "Red" second = "Blue" # جا به جایی در یک خط! first, second = second, first print(first) # خروجی: Blue print(second) # خروجی: Red

🔹 حذف یک متغیر

temp = "I will be deleted" print(temp) # خروجی: I will be deleted del temp # حالا اگه print(temp) بزنی، خطای NameError می‌گیری!

🧪 تمرین‌های عملی متغیرها

روی هر کارت کلیک کن تا پاسخ را ببینی! (اول خودت حل کن، بعد جواب رو چک کن!)

۱
اسمت رو توی متغیر name بریز و چاپ کن
👆 کلیک کن تا جواب رو ببینی
✅ پاسخ:
name = "Amir" print(name) # خروجی: Amir
۲
دو عدد رو در متغیر بریز و جمعشون رو چاپ کن
👆 کلیک کن تا جواب رو ببینی
✅ پاسخ:
a = 15 b = 35 print(a + b) # خروجی: 50
۳
مقدار متغیر x رو از 10 به 20 تغییر بده و چاپ کن
👆 کلیک کن تا جواب رو ببینی
✅ پاسخ:
x = 10 print(x) # 10 x = 20 print(x) # 20
۴
با f-string اسم و سنت رو در یک جمله چاپ کن
👆 کلیک کن تا جواب رو ببینی
✅ پاسخ:
name = "Sara" age = 22 print(f"My name is {name} and I am {age}") # خروجی: My name is Sara and I am 22
۵
طول و عرض مستطیل رو در متغیر بریز و مساحتش رو چاپ کن
👆 کلیک کن تا جواب رو ببینی
✅ پاسخ:
length = 10 width = 5 area = length * width print(area) # خروجی: 50
۶
مقادیر دو متغیر a و b رو با هم عوض کن
👆 کلیک کن تا جواب رو ببینی
✅ پاسخ:
a = "First" b = "Second" a, b = b, a print(a) # Second print(b) # First
