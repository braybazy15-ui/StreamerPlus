🔀 شرط یعنی چی؟ چرا نیاز داریم؟

تا حالا برنامه‌هایی که نوشتیم خطی بودند. یعنی از بالا به پایین اجرا می‌شدند بدون هیچ تصمیم‌گیری. اما در دنیای واقعی، برنامه‌ها باید تصمیم بگیرند! مثلاً: «اگر رمز عبور درست بود، وارد شو»، «اگر نمره بالای ۱۰ بود، قبول شدی»، «اگر هوا آفتابی بود، برو پارک وگرنه در خانه بمان».

شرط (Condition) یعنی از برنامه بپرسی: «آیا فلان چیز درست است؟» اگر جواب بله (True) بود، یک کار انجام بده، اگر خیر (False) بود، کار دیگری انجام بده (یا هیچ کاری نکن).

🧠 مثال واقعی: تصور کن پشت درِ یک باشگاه ورزشی ایستاده‌ای. نگهبان می‌گوید: «اگر سنت بالای ۱۸ است، می‌توانی وارد شوی، وگرنه نمی‌توانی.» این دقیقاً یک شرط است! برنامه هم همین کار را می‌کند.

🧮 عملگرهای مقایسه‌ای (Comparison Operators)

برای نوشتن شرط، باید بتوانی مقایسه کنی. مثلاً بپرسی: آیا سن از ۱۸ بزرگتر است؟ آیا نمره با ۲۰ برابر است؟ برای این کار از عملگرهای مقایسه‌ای استفاده می‌کنیم. این عملگرها همیشه True یا False برمی‌گردانند.

عملگرمعنیمثالنتیجه
==برابر بودن5 == 5True
!=نابرابر بودن5 != 3True
>بزرگتر بودن10 > 5True
<کوچکتر بودن3 < 8True
>=بزرگتر یا مساوی5 >= 5True
<=کوچکتر یا مساوی4 <= 5True
⚠️ تفاوت = و == :
= یعنی مقداردهی (مقدار را در متغیر بریز)
== یعنی مقایسه (ببین آیا برابرند؟)
این دو را قاطی نکن! رایج‌ترین اشتباه دنیای برنامه‌نویسی!

✍️ ساختار شرط if

ساده‌ترین شکل شرط:

if شرط: # کدی که فقط در صورت درست بودن شرط اجرا می‌شود print("شرط برقرار بود!")
age = int(input("سن تو چند ساله؟ ")) if age >= 18: print("تو می‌تونی رأی بدی!") print("این خط همیشه اجرا می‌شود.")
سن تو چند ساله؟ 20 تو می‌تونی رأی بدی! این خط همیشه اجرا می‌شود.

✍️ شرط if-else (اگر-وگرنه)

وقتی می‌خواهی اگر شرط درست بود یک کار، وگرنه کار دیگر انجام شود:

score = int(input("نمره: ")) if score >= 10: print("🎉 قبول شدی!") else: print("😔 رد شدی.")
نمره: 15 🎉 قبول شدی!

✍️ شرط if-elif-else (چند شرطی)

وقتی چند حالت مختلف داری:

score = int(input("نمره: ")) if score >= 18: print("⭐ عالی!") elif score >= 14: print("👍 خوب!") elif score >= 10: print("👌 قبول!") else: print("😔 رد شدی.")
نمره: 16 👍 خوب!

📐 فرورفتگی (Indentation) در پایتون

خیلی مهم: در پایتون، کدهایی که داخل if قرار می‌گیرند، باید با فاصله (تورفتگی) نوشته شوند. معمولاً ۴ فاصله یا یک Tab. اگر فرورفتگی را رعایت نکنی، خطا می‌گیری!

if True: print("این خط ۴ فاصله جلوتر است") # ✅ درست print("این خط هم داخل if است") # ✅ درست print("این خط خارج از if است") # بدون فاصله = خارج از if
